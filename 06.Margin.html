CSS Margin

CSS margin is used to create space outside an element's border. It helps separate elements from each other and improves the overall layout of a web page.

Why Use Margin?

Margins make a webpage look clean and organized by adding empty space around elements. Without margins, content may appear crowded and difficult to read.

Types of Margins

CSS provides four margin properties: margin-top, margin-right, margin-bottom, and margin-left. You can also use the shorthand margin property to set all four margins at once.

Example

margin: 20px; adds a 20px margin on all sides.

margin: 10px 20px; applies 10px to top & bottom and 20px to left & right.

Conclusion

Margins improve spacing, readability, and the overall appearance of web pages. They are one of the most important CSS properties used in layout design.
