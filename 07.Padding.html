CSS Padding

CSS padding is used to create space between an element's content and its border. It makes the content appear more spacious and improves the overall design of a webpage.

Why Use Padding?

Padding prevents text and other content from touching the border of an element. It creates comfortable spacing inside the element, making the content easier to read.

Types of Padding

CSS provides four padding properties: padding-top, padding-right, padding-bottom, and padding-left. You can also use the shorthand padding property to set all four values at once.

Example

padding: 20px; adds 20px of space between the content and the border on all four sides.

padding: 10px 20px; applies 10px to the top & bottom and 20px to the left & right.

Conclusion

Padding improves the appearance of elements by creating space inside the border. It enhances readability and gives a clean, professional layout to web pages.
